# Front-end — O Quiz da Livinha

React + Vite.

## Rodando localmente

```bash
npm install
cp .env.example .env
# edite VITE_API_URL apontando pro seu backend local ou em produção

npm run dev
```

## Build de produção

```bash
npm run build
```

## Deploy no Vercel

1. Importe este repositório (ou esta pasta, como Root Directory) no Vercel.
2. Configure a variável de ambiente `VITE_API_URL` com a URL do backend.
3. Deploy — o Vercel detecta o Vite automaticamente.

Veja o passo a passo completo no `README.md` da raiz do projeto.
